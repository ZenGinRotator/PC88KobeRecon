-- In Rom Disk Name meanings --
Sets & Alts (in square brackets [])
Set 1 - original release of rom
Set 2, ..., Set n - nth release of rom on disks
Alt 1 - alternate version with possible hacks, cheat codes, bug fixes after initial rom release on (Set 1) disks
Alt 2 - subsequent of an alternatie release of rom on disk.
