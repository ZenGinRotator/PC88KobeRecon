    set "!wrd_tail=%%i"
)

    set "wrd=%%i"
)

    rem This is a demo-disk rom.
)

-- parse 


(Disk ...) - from parentheses, implies multidisc -> id -> "Disk " (with space)

Given rom R, 
- R might be single disk only
- R might be multi disk only
- R might include both single and multi-disk versions

Given rom R that is a single-disk version
- R might have a single release ("" or Set 1)
- R might be a re-release (Set 1, Set 2, Set 3, ... Set N)
- R might be an alternate version (Alt 1, [a], Alt 2, Alt 3, ..., Alt n)
- R might be a re-release or a revised, alternate version ( Set 1 w/ Alt 2, Alt 3, ..., Alt n)

Given rom R that is a multi-disk version (Disk A, Disk 1, Disk 01, Disk B, Disk 2, Disk 02, etc.)
- R might be a single release ("" or Set 1)
- R might be a re-release (Set 2, Set 3, ... Set N)
- R might be an alternate version (Alt 1, [a])
- R might be a re-release or a revised, alternate version (Alt 2, Alt 3, ..., Alt n)

Given rom R
roms/pc88/R/
    Single Disk
        1st ed.
            Set-free label
            Set 1
        Re-release
            Set 1, Alt 1, Set 1 w/ Alt
